Features

Seriously confidential

Your data is encrypted before it reaches the cloud, so neither the network, the cloud, nor Least Authority can ever read your data.

Robustly architected

Provider-independent security means if one server or cloud provider goes down, your data is still safe and available.


Open source!

Both the client-side and server-side software are based on the LAFS free and open source software project.

Easy to use

If you are familiar with the command line and can write a cron job, Least-Authority Backup will be a snap.
